# Price Comparison System
A web-based price comparison tool built with Python and Streamlit that fetches real-time product prices from multiple online stores using the SerpApi Google Shopping API.

## 📌 Features

* 🔍 Search any product by name
* 💰 Compare prices from multiple sellers (Amazon, Flipkart, Meesho, etc.)
* 🏆 Automatically finds the best (lowest) price option
* 🔗 Direct buy links to product pages
* 📊 Bar chart and Pie chart for visual price comparison
* 🖼️ Product thumbnail preview in sidebar

## How to Use

* Enter a product name in the sidebar (e.g. boAt headphones)
* Enter the number of options you want to compare
* Click Compare Prices
* View results, buy links, and charts
